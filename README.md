# filekeep

Verify integrity and track file changes.

## Install / Uninstall

    # Install
    git clone https://github.com/goncalomb/filekeep.git
    pip3 install filekeep/

    # Install for Development (editable)
    git clone https://github.com/goncalomb/filekeep.git
    pip3 install -e filekeep/

    # Uninstall
    pip3 uninstall filekeep
